# twins-kdb-fetch-notify

KDBの差分JSONを取得し、科目の追加・変更・削除をDiscordのWebhookで通知します。通知は1メッセージにまとめ、科目ごとにembedを分けて送信します。

## 出力するembedの形式

1科目につき1つのembedを作成し、以下の情報を含めます。

- **タイトル**: `科目名 (科目コード)`
- **説明**: 差分の種別（added / modified / removed）
- **フィールド**:
  - **学期**: 例) 春学期 (A)
  - **担当教員**: 例) 常 瀟琳
  - **所属**: 例) 人文・文化学群
  - **対象年次**: 例) 1年
  - **時間割**: 例) 春A(金2), 春B(金2)
  - **変更内容** (modifiedのみ): 変更前/変更後の要約

色は差分種別で分けます。

- added: 緑
- modified: 黄
- removed: 赤
